# Diseño y Desarrollo Web

Este repositorio contiene el proyecto correspondiente a la asignatura **Diseño y Desarrollo Web** de UADE, a cargo de la profesora Mg. Patricia Litovicius.

## Estructura
- tarea-3: portfolio personal con HTML y CSS, unificado en un solo proyecto.

## Requisitos y herramientas
Para trabajar con el proyecto se recomienda:
- Editor de codigo: Visual Studio Code.
- Control de versiones y publicacion: GitHub.
- Otras herramientas: plataformas institucionales (Microsoft Teams, Web UADE) y herramientas de inteligencia artificial.
